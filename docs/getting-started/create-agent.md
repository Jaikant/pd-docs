
# Create Chatbot Agent
**Estimated Time: 1 minute**

### Sign In
- Sign in to [Predictable Dialogs](https://predictabledialogs.com/sign-in).

### Go to page which creates agents
- Click the card with **+ Create new agent**. 
- Enter an agent name, this creates your chatbot agent.
- Click "+ Add AI" button and enter your system instructions.
- Add files which you want your chatbot to answer from.

You chatbot agent is now ready with your system instruction.
