# Add a Phone Number
**Estimated Time: 4 minutes**

We need to add your business phone number in the Meta console to connect it with your WhatsApp Business account. Once added, Meta generates a unique Phone Number ID. You'll use this ID in the Predictable Dialogs app to enable sending and receiving messages through WhatsApp Cloud API.

Below are the steps to add your phone number:


## Access Your App Settings
- Visit [Facebook Developers](https://developers.facebook.com/apps).
- Select the app you wish to configure.

## Navigate to WhatsApp Configuration
- Click on "Customize the connect with customers through WhatsApp use case"
![Start API Image](/img/whatsapp/connect-whatsapp.webp)


- Click on "Start using the API"
![Start API Image](/img/whatsapp/start-using-api.webp)


- Scroll down to the section **Send and receive messages**. Click on the **From** number and select **+ Add phone number** (Image below)
![Start API Image](/img/whatsapp/send-receive-messages.webp)


- Add the "WhatsApp Business display name", Category and Description 
  ![Start API Image](/img/whatsapp/Whatsapp-business-display-name.webp)

- Then Add your WhatsApp phone number 
  ![Start API Image](/img/whatsapp/add-phone.webp)

## Copy Your Phone Number ID
- Once the number is added, ensure to copy the “Phone Number ID” associated with the newly added phone number. You’ll need this ID for API calls.
  To get the phone number id, select the newly added number from the "From" drop down. The "Phone number id" will be displayed below the input once selected.
