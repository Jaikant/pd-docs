# Avatars

Update the avatar image shown for your agent in the chat widget.

## Using the Theme Dashboard

1. Get the URL of the image you want to use for the avatar.
2. Go to your agent in the Predictable Dialogs app.
3. Open the **Theme** page.
4. Select the **Avatars** tab.
5. Click the toggle next to the **Avatar** label to enable a custom avatar.
6. Click the profile circle and paste the image URL.

The preview will update to show the new avatar.
