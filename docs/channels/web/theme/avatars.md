# Avatars

Use the **Avatars** tab to set both AI and User avatars shown in the chat header and message bubbles.

## Configure in Live Theming

1. Go to your agent in the Predictable Dialogs app.
2. Open the **Theme** page.
3. Select the **Avatars** tab.
4. Choose whether you want to edit the **AI avatar** or **User avatar**.
5. Pick an icon/image using **Upload**, **Emoji**, or **Link**.
6. If you use **Link**, click the page **Save** button (or Save FAB) to persist URL edits.

Upload and Emoji selections are saved immediately. The preview updates instantly.

![Theme Customization Page with Avatar Settings](/img/theming/theme-page.webp)

## Supported image formats

- SVG
- PNG
- JPG/JPEG
- WebP
- GIF

File size limit: **2 MB**
